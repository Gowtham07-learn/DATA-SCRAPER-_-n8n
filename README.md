Data Scraper using n8n
Description

This project uses n8n to automatically scrape data from web pages and extract structured information such as article titles, image URLs, publish dates, and metadata. The workflow leverages JavaScript code nodes to parse HTML content and transform it into clean, usable JSON data.

Features

Automated web scraping using n8n workflows
Extracts key article details (title, image, date, links)
Custom JavaScript parsing logic
Easy to extend for APIs, databases, or notifications

Use Cases

News aggregation
Content monitoring
Data collection for analytics
Automation pipelines

Requirements

n8n (self-hosted or cloud)
Basic JavaScript knowledge
Target webpage HTML structure
